# YT-DOWNLOAD

## Execução da API:
### Pelo terminal, acesse a pasta ```ytdl-api``` e execute os seguintes comandos:
~~~console
foo@bar$ npm install
foo@bar$ node index.js
~~~

## Execução da aplicação:
### Pelo terminal, acesse a pasta ```ytdl-app``` e execute os seguintes comandos:
~~~console
foo@bar$ npm install
foo@bar$ npm start
~~~
